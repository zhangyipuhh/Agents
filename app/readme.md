# App 文件夹说明

## 项目概述
本文件夹包含智能体服务应用程序的核心代码和资源文件。

## 程序入口
- **MainServer.py**：应用程序的启动入口，负责初始化并启动整个服务系统

## 目录结构

### agents/
- **用途**：存储主智能体相关文件
- **说明**：包含核心智能体的逻辑实现和主要功能模块

### config/
- **用途**：统一配置文件存储目录
- **说明**：为 routers 文件夹、test 文件夹和 mainserver.py 提供配置支持，集中管理所有配置信息

### routers/
- **用途**：智能体服务路由模块
- **说明**：包含各种智能体服务的具体实现，通过 mainserver.py 注册到主页面，提供 API 接口

### test/
- **用途**：测试文件存储目录
- **说明**：存放功能测试和智能体测试相关的代码文件

### data/
- **用途**：数据持久化存储目录
- **说明**：存储 routers 文件夹和 test 文件夹中各功能模块运行时产生的数据文件
