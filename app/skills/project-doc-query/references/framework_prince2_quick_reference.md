# PRINCE2 Quick Reference (1 page)

> **UK Government framework** · 7 Principles / 7 Themes / 7 Processes · Controlled project · For "how to do it specifically"

## 7 Principles (all must be followed; none may be omitted)

| # | Principle | Chinese | Meaning |
|---|---|---|---|
| 1 | Continued business justification | 持续业务验证 | Verify business justification at every stage |
| 2 | Learn from experience | 吸取经验教训 | Continuous learning |
| 3 | Defined roles and responsibilities | 明确定义的角色与责任 | RACI-style |
| 4 | Manage by stages | 按阶段管理 | Staged checkpoints |
| 5 | Manage by exception | 偏差管理 | Set tolerance; exceedances escalate |
| 6 | Focus on products | 聚焦产品 | Define the product first, then do the work |
| 7 | Tailor to suit the project | 裁剪适应项目 | Adjust for project size / complexity |

## 7 Themes (continuously running areas of focus)

| # | Theme | Core |
|---|---|---|
| 1 | Business Case | Business case: why to do it |

| 5 | Risk | Risk: what to do |

## 7 Processes

| # | Process | Chinese | When |
|---|---|---|---|
| 1 | SU | 项目准备 (Starting Up) | Before project startup |
| 2 | DP | 项目指导 (Directing) | Board level |
| 3 | IP | 项目启动 (Initiating) | Project formally established |
| 4 | CS | 阶段边界控制 (Controlling a Stage) | Within each stage |
| 5 | MP | 阶段管理 (Managing Product Delivery) | Team-level execution |
| 6 | SB | 阶段边界 (Managing Stage Boundaries) | Switching between stages |
| 7 | CP | 项目收尾 (Closing a Project) | Project ends |

## Suite Usage Recommendations

- User asks "how to do it specifically" → use the PRINCE2 landing layer
- User asks "roles / responsibilities" → Organization theme + Process CS
- User asks "how is change controlled" → Change theme
- User asks "how is risk managed" → Risk theme

## Key Terminology

| Term | Chinese | Meaning |
|---|---|---|
| Business Case | 商业论证 | The "why" of the project; reviewed at every stage |
| Tolerance | 容差 | Acceptable range of plan deviation (time / cost / scope / quality / risk / benefit) |
| Exception | 偏差升级 | Exceed tolerance → escalate to the board |
| Highlight Report | 阶段亮点报告 | Periodic report within a stage |
| Lessons Log | 经验教训日志 | Continuously recorded |
| Product Description | 产品描述 | Core deliverable definition of the "Focus on products" principle |
