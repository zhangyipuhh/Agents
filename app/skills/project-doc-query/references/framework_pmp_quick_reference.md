# PMP Quick Reference (1 page)

> **PMI framework** · 5 Process Groups / 10 Knowledge Areas / 49 Processes · International standard · For "management system overview"

## 5 Process Groups

| Process Group | Chinese | Core Question |
|---|---|---|
| Initiating | 启动 (Initiating) | Project authorization, stakeholder identification |
| Planning | 规划 (Planning) | Develop the action route |
| Executing | 执行 (Executing) | Complete the project management plan |
| Monitoring & Controlling | 监控 (Monitoring & Controlling) | Track, review, adjust |
| Closing | 收尾 (Closing) | Formally end the project or phase |

## 10 Knowledge Areas

| # | Knowledge Area | Key Outputs |
|---|---|---|
| 1 | Project Integration Management | Project Charter, Project Management Plan |
| 2 | Project Scope Management | Scope Statement, WBS, Scope Baseline |
| 3 | Project Schedule Management | Schedule Baseline, Schedule |
| 4 | Project Cost Management | Cost Baseline, Budget |
| 5 | Project Quality Management | Quality Plan, Quality Report |
| 6 | Project Resource Management | Team Building, Resource Allocation |
| 7 | Project Communication Management | Communication Management Plan |
| 8 | Project Risk Management | Risk Register, Risk Response |
| 9 | Project Procurement Management | Procurement Contract, Bidding Documents |
| 10 | Project Stakeholder Management | Stakeholder Register, Expectation Management |

## 49 Processes (distributed by Process Group × Knowledge Area)

> The suite does not require rote memorization; just index by the two dimensions "Process Group / Knowledge Area".

## Suite Usage Recommendations

- User asks "what does the project management system look like" → use the PMP framework layer
- User asks "what should I do" → use PMP to find the corresponding knowledge area
- User asks "how to do it specifically" → switch to the PRINCE2 landing layer

## Key Terminology

| Term | Chinese | Meaning |
|---|---|---|
| WBS | 工作分解结构 (Work Breakdown Structure) | Tasks decomposed into manageable units |
| RACI | 责任分配矩阵 (Responsibility Assignment Matrix) | Responsible / Accountable / Consulted / Informed |
| EVM | 挣值管理 (Earned Value Management) | PV / EV / AC three indicators |
| CCB | 变更控制委员会 (Change Control Board) | Approve major changes |
| SIPOC | 供应商-输入-过程-输出-客户 (Suppliers-Inputs-Process-Outputs-Customers) | Process scope definition |
