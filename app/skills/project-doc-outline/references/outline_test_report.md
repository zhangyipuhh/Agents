# Test Report — Outline Template

> Reference source: 项目根目录/03_技术文档及评审/05_测试、验收/ — historical test reports under that directory

1. **Introduction**
   - 1.1 Purpose
   - 1.2 Scope
   - 1.3 Reference Materials
2. **Test Overview**
   - 2.1 Object Under Test

   - 2.3 Test Time
3. **Test Basis and Criteria**
4. **Summary of Test Results**
   - 4.1 Test Case Statistics

5. **Detailed Test Results**
   - 5.1 Functional Testing
   - 5.2 Performance Testing
   - 5.3 Security Testing

   - 5.5 Acceptance Testing
6. **Defect Analysis**
   - 6.1 Fixed Defects
   - 6.2 Unfixed Defects
   - 6.3 Open Issues
7. **Test Conclusion and Recommendations**
8. **Appendix**
   - 8.1 Test Case List
   - 8.2 Defect List
   - 8.3 Test Data
