# Other Process Documents — Outline Template

> Applies to: Weekly reports, meeting minutes, checklists, risk registers, and other process documents
> Review: Usually no formal review is required

## Weekly Report Outline

1. **Work Completed This Week**
2. **Work Plan for Next Week**
3. **Existing Issues and Risks**
4. **Items Requiring Coordination**
5. **Key Data** (progress / cost / defects)

## Meeting Minutes Outline

1. **Meeting Basic Information** (time / location / host / recorder / attendees / absentees)
2. **Topics and Discussion**
3. **Resolutions**
4. **Action Items** (responsible person / deadline)
5. **Open Issues**

## Checklist Outline

1. **Inspection Scope**
2. **Inspection Basis**
3. **Inspection Items** (ID / item / criterion / result / remark)
4. **Inspection Conclusion**
5. **Rectification Suggestions**

## Risk Register Outline

1. **Risk Overview**
2. **Risk Items** (ID / description / probability / impact / response / owner / status)
3. **Risk Trend**
4. **Closed Risks**
