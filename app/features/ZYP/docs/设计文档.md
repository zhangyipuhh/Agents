# ZYP TUI Agent 设计文档

## 1. 设计目标

- **模块化**: 各组件职责分离，便于维护和测试
- **可扩展**: 支持新增模型类型和工具
- **轻量**: 最小化依赖，适合打包为 exe
- **跨平台**: 支持 Windows/Linux/macOS

## 2. 技术架构

### 2.1 层次结构

```
┌─────────────────────────────────────────────────────────────┐
│                    Presentation Layer (TUI)                │
│  ZYPTuiApp → Sidebar / ChatArea / InputArea               │
└─────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────┐
│                    Business Logic Layer                     │
│  ZYPAgent → SessionHandler / ModelHandler                  │
└─────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────┐
│                    Data Access Layer                         │
│  SessionStorage / ConfigStorage                             │
└─────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────┐
│                    Framework Layer                          │
│  app.core.agent.Agent (LangGraph)                          │
│  ModelFactory → deepseek / ollama / openai                 │
└─────────────────────────────────────────────────────────────┘
```

### 2.2 核心类职责

| 类 | 层级 | 职责 |
|----|------|------|
| `ZYPTuiApp` | Presentation | TUI主应用，布局管理，事件分发 |
| `Sidebar` | Presentation | 会话列表，模型选择 |
| `ChatArea` | Presentation | 消息显示，Markdown渲染 |
| `InputArea` | Presentation | 多行输入，发送按钮 |
| `ZYPAgent` | Business | Agent封装，流式调用 |
| `SessionHandler` | Business | 会话CRUD，状态管理 |
| `ModelHandler` | Business | 模型切换配置 |
| `SessionStorage` | Data | 会话持久化 |
| `ConfigStorage` | Data | 配置持久化 |
| `ZYPConfig` | Data | 配置数据结构 |

## 3. 数据流设计

### 3.1 消息发送流程

```
InputArea [用户按 Ctrl+Enter]
      │
      ▼
ZYPTuiApp.on_message_submit(text)
      │
      ▼
SessionHandler.add_message(session_id, user_message)
      │
      ▼
ChatArea.append_message(user_message)
      │
      ▼
ZYPAgent.chat(message, session_id, stream_callback)
      │
      ├──► LangGraph Agent 处理
      │
      └──► stream_callback(chunk) ──► ChatArea.append_chunk()
```

### 3.2 模型切换流程

```
Sidebar.on_model_select(model_type)
      │
      ▼
ModelHandler.show_config_form(model_type)
      │
      ▼
用户填写配置并确认
      │
      ▼
ModelHandler.validate_config(config)
      │
      ▼
ModelHandler.create_agent_config(config)
      │
      ▼
ZYPAgent.reinitialize(new_config)
      │
      ▼
SessionHandler.update_session_model(session_id, config)
```

## 8. 流式输出设计

### 8.1 流式接口

```python
async def chat(self, message: str, session_id: str,
               stream_callback: Callable[[str], None]) -> str:
    """流式聊天"""
    
    config = ZYPExecuteConfig(
        configurable=ZYPConfigurableConfig(thread_id=session_id)
    )
    
    state = ZYPState(
        messages=[message],
        error_limit=5,
        limit=25,
        file_chunk_read_progress=1,
    )
    
    context = ZYPContext(
        session_id=session_id,
        store_id="default",
    )
    
    full_response = ""
    
    async for chunk in self._agent.astream(
        config=config,
        input_state=state,
        context=context
    ):
        if chunk.get("messages"):
            last_msg = chunk["messages"][-1]
            if hasattr(last_msg, "content"):
                content = last_msg.content
                # 计算新增内容
                delta = content[len(full_response):]
                full_response = content
                await stream_callback(delta)
                
    return full_response
```

## 10. 错误处理设计

### 10.1 错误类型

```python
class ZYPError(Exception):
    """基础异常"""
    pass

class NetworkError(ZYPError):
    """网络错误"""
    pass

class AuthError(ZYPError):
    """认证错误"""
    pass

class TimeoutError(ZYPError):
    """超时错误"""
    pass

class StorageError(ZYPError):
    """存储错误"""
    pass
```

### 10.2 错误恢复策略

| 错误类型 | 策略 |
|---------|------|
| NetworkError | 重试3次，指数退避 |
| AuthError | 提示检查API Key |
| TimeoutError | 提示超时，可重试 |
| StorageError | 创建新存储，警告用户 |
