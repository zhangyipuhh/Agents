# ZYP TUI Agent 功能文档

## 1. 项目概述

**项目名称**: ZYP TUI Agent  
**版本**: 1.0.0  
**类型**: 终端用户界面 (TUI) 聊天智能体应用  
**核心功能**: 支持多模型切换的对话智能体，可调用工具，打包为 Windows exe

## 4. 数据结构

### 4.1 会话数据结构

```python
class Session:
    id: str                      # 会话唯一ID (UUID)
    name: str                    # 会话名称 (可编辑)
    created_at: datetime         # 创建时间
    updated_at: datetime         # 更新时间
    messages: list[Message]     # 消息列表
    model_config: ModelConfig   # 当前模型配置

class Message:
    id: str                      # 消息ID
    role: str                    # "user" / "assistant" / "system" / "tool"
    content: str                  # 消息内容
    tool_calls: list[dict]       # 工具调用信息 (可选)
    created_at: datetime         # 创建时间

class ModelConfig:
    model_type: str              # "deepseek" / "ollama" / "openai"
    model_name: str              # 具体模型名
    api_key: str                 # API密钥
    base_url: str                # API地址
    temperature: float           # 温度参数
```

### 4.2 存储文件结构

```
ZYP/
└── data/
    └── sessions/
        ├── sessions.json        # 会话列表索引
        └── session_{id}.json   # 各会话详情
```

## 5. 错误处理

| 错误场景 | 处理方式 |
|---------|----------|
| 网络请求失败 | 显示错误提示 "网络请求失败，请检查连接"，保留用户输入 |
| API Key 无效 | 显示 "API Key 无效，请检查配置" |
| 模型超时 | 重试3次后显示 "请求超时，请稍后重试" |
| 会话文件损坏 | 创建新会话，显示警告 "会话文件损坏，已创建新会话" |
| 终端太小 | 显示提示 "终端窗口太小，请调大后重试" |
| 空输入提交 | 禁用发送按钮，防止空提交 |

## 6. 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Enter` | 发送消息 |
| `Ctrl+C` | 中断AI生成 |
| `Ctrl+N` | 新建会话 |
| `Ctrl+,` | 打开设置 |
| `Esc` | 取消当前操作 |

## 7. 配置项

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| default_model | "deepseek" | 默认模型类型 |
| default_model_name | "deepseek-chat" | 默认模型名称 |
| temperature | 0 | 默认温度 |
| max_tokens | 20000 | 最大Token数 |
| session_storage_path | "./data/sessions" | 会话存储路径 |
| window_width | 120 | 默认窗口宽度 |
| window_height | 40 | 默认窗口高度 |
